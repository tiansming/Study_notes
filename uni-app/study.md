# 缓存

###### 存储数据

```
uni.setStorage
```

###### 获取数据

```
uni.getStorage({
	key:"id",
	success(){
	console.log("获取成功")
	}
})
```

###### 删除数据

```
uni.removeStorage({
	key:"id",
	success(){
	console.log("删除")
	}
})
```
